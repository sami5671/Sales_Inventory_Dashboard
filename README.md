# Sales & Inventory Dashboard
This Streamlit-based Sales & Inventory Dashboard is a simple yet powerful tool for managing and visualizing product sales and stock data. Users can add new sales entries, view key inventory metrics such as total stock, sold units, and remaining stock, and analyze trends with interactive line and pie charts. The dashboard allows filtering by product and visual comparison through dynamic pie charts, showcasing top-selling items or selected products. A full data table provides a clear view of all records, enhancing data transparency. With its clean interface and visual insights, this dashboard helps small businesses, shops, or inventory managers track performance and make data-driven decisions efficiently. It‚Äôs easy to use, customizable, and supports CSV-based data storage for persistent records.

- Sales Entry Form ‚Äì Add new product sales with stock, sold quantity, and date through an intuitive form.

- Inventory Metrics ‚Äì View real-time stats like total stocked, sold, and remaining units.

**Python:** Core programming language used for data handling and logic.

**Matplotlib:** For generating line and pie charts to visualize sales trends.
